# Coffee Shop POS

Sistem Point of Sales (POS) berbasis web yang dirancang khusus untuk coffee shop skala menengah. Project ini dibangun menggunakan arsitektur bersih (*clean architecture*) dengan bahasa pemrograman Go (Golang) dan memfasilitasi kebutuhan operasional dari inventori (stok), manajemen transaksi, sampai dengan integrasi pembayaran.

Selengkapnya mengenai detail spesifikasi dan desain database dapat dilihat pada:
- `PRD.md` - Dokumen Product Requirements.
- `SCHEMA.md` - Dokumen detail relasi tabel Database MySQL.
- `API_CONTRACT.md` - Kontrak API endpoint.
